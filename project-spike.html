Spike Originator Model

A hybrid fraud detection engine combining anomaly detection with supervised claim-level classification to enhance operational fraud detection in healthcare systems.

Problem

The legacy system flagged providers using unsupervised anomaly detection but lacked claim-level precision, resulting in operational inefficiencies.

Solution

Integrated supervised ML classification layer over anomaly signals. Implemented claim-level scoring and dollar threshold prioritization. Established quarterly retraining pipeline.

Claims Data
Feature Engineering
Anomaly Detection
Supervised Model
Risk Scoring

Impact

• 11% precision improvement • ~$2.7M annual financial impact • 23K+ monthly claims processed • Automated governance dashboards
